    Quick help

    How to use

    Pick Day A/B/C, tap an exercise, then press Start. In normal mode, tap Complete set after each set to start the rest timer. In Interval mode, Start runs work→rest automatically.

    Best default

    Start in normal mode with 60–90s rest. Use Interval mode later for conditioning (e.g., 40s work / 20s rest).

    Add to Home Screen

    On iPhone/iPad: open this in Safari → Share → Add to Home Screen. On Mac: open the folder and double-click index.html.
